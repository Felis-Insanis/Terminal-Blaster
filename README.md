# Game

## How To Run The Game

### Movement
- W to go UP
- S to go DOWN
- A to go LEFT
- D to go RIGHT
### Shooting
Press SPACE to shoot. <br>
You can't move while shooting.
### Exiting the game
To exit the game, press ESC or any of the ARROW KEYS
